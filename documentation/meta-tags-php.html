Meta Tags Php (Mtp)

Uma classe para auxiliar na criação de meta tags de redes sociais e meta tags seo.


Instalção Como Usar Tags Open Graph Tags SEO
Intalação composer
   
    $ composer require kelvysmoura/mtp
Como usar com composer
    require 'vender/autoload.php';
    use MetaTagsPhp\Mtp;
    $metatags = new Mtp();
Gerando tags OpenGraph do facebook
    use MetaTagsPhp\Mtp;
    $metatags = new Mtp();

    # Gerando tags html
    $metatags->OpenGraph([
      "title" => "Meu Site"
    ]);

    # Mostrando Tags
    $metatags->showTags(); 
    // resultado: <meta property="og:title" content="Meu Site" />
